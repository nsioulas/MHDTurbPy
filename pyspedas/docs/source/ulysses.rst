Ulysses
========================================================================
The routines in this module can be used to load data from the Ulysses mission.


Magnetic field (VHM)
----------------------------------------------------------
.. autofunction:: pyspedas.ulysses.vhm

Example
^^^^^^^^^

.. code-block:: python
   
   import pyspedas
   from pytplot import tplot
   vhm_vars = pyspedas.ulysses.vhm()
   tplot('B_MAG')

.. image:: _static/ulysses_vhm.png
   :align: center
   :class: imgborder
